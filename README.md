Gerador de Senhas em Python ->

Este é um script simples que criei para praticar lógica de programação e manipulação de strings em Python. Ele gera senhas aleatórias combinando letras, números e símbolos.
O que ele faz:

    Deixa você escolher o tamanho da senha.
    
    Mistura caracteres maiúsculos, minúsculos, números e sinais.

    Roda direto no terminal.

Como rodar:

Se você já tem o Python instalado:

    Clone o repo: git clone https://github.com/leeocomL/Gerador-de-senhas.git.

    Entre na pasta e execute: python main.py.

Sobre a segurança:

Como este projeto é para estudo, usei a biblioteca random. Se for usar para senhas reais e importantes, o ideal é trocar para a biblioteca secrets, que é mais segura contra ataques.
